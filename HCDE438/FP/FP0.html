Final Project Proposal

One sentence of the project

The project will be a food blog website that comes with a backend platform for people to upload their blog content to update the contents on the blog page.

Drawings/Design

Description of the two versions

Prototype version

    The project will need to be functional in this stage of development. The functionalities needed to be completed include:

  1. Allow users to input blog content from the content management page
  2. The website updates the contents by getting the data from the backend
  3. A backend for storing the data/contents

Final version

    The final version of the project will need to be similar to the drawings aesthetically. Some steps to reach the final version include:

  1. Using React to build the user interface
  2. Connect the component with the functions from the prototype version
  3. Conduct a few user testing for iteration
